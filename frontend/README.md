A fast typing test for programmers. Practice typing real code in multiple languages.

## Features

- 7 programming languages (JavaScript, TypeScript, Python, Java, C++, Go, Rust)
- Timer mode (15s, 30s, 60s) or complete code mode
- Real-time WPM, CPM, and accuracy tracking
- Clean, distraction-free interface

## Installation

```bash
git clone https://github.com/Raynzler/SyntaxRacer.git
cd SyntaxRacer
npm install
npm run dev

```
Open http://localhost:3000
## How to Use     

- Pick a language
- Choose timer or code mode
- Start typing when ready
- View your stats

## Shortcuts:

- Shift + Enter - New test
- Esc - Reset test

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
